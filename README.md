# mkswap

This role is used for quick creation and mounting swap file.

## Getting Started

No parameters needed to set beforehand. Just run the role like so:

```ansible-playbook -i '<IP>,' playbooks/mkswap.yml```

### Prerequisites

No prerequisites required
